# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅ |

## Reporting a Vulnerability

This project handles A2A protocol communication which may involve
agent-to-agent authentication tokens. If you discover a security issue:

1. **Do NOT** open a public GitHub issue
2. Send details to [maisonnat@gmail.com](mailto:maisonnat@gmail.com)
3. Include as much context as possible:
   - Version affected
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

You should receive a response within 48 hours. If not, please follow up.

## Scope

- A2A protocol implementation
- Authentication mechanisms (bearer tokens)
- Task data handling
- Plugin registration and execution

## Out of Scope

- Hermes Agent core vulnerabilities (report to [Nous Research](https://github.com/NousResearch/hermes-agent/security))
- OpenCode vulnerabilities (report to [OpenCode](https://github.com/anomalyco/opencode/security))
- A2A protocol specification issues (report to [A2A Project](https://github.com/a2aproject/A2A))
